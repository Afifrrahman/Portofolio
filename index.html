Lms Inovindo Academy

LMS Inovindo merupakan platform pembelajaran yang terintegrasi dengan sistem utama melalui mekanisme Single Sign-On (SSO) berbasis JSON Web Token (JWT), di mana pengguna cukup melakukan satu kali login pada website utama Inovindo untuk kemudian dapat mengakses LMS tanpa perlu autentikasi ulang.
